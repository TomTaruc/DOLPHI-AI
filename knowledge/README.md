# Knowledge Base

Put permanent school chatbot documents in this folder.

Supported files:
- PDF
- DOCX
- TXT
- MD
- CSV
- XLSX
- PPTX

Example:
knowledge/mapua handbook.pdf
knowledge/REFUNDS AND RETURN POLICY.pdf

Files in this folder are used as the global school Knowledge Base for DOLPHI AI.
Do not commit private or copyrighted school documents unless you are allowed to.
